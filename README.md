# Peggy 🎨
**Peggy** är ett lättviktigt, webbaserat verktyg för att förvandla dina favoritbilder till färdiga pärlmönster. Appen körs helt lokalt i din webbläsare – inga servrar, ingen uppladdning av personlig data, bara ren kreativitet.

## ✨ Funktioner
* **Pixelering:** Omvandla fotografier till ett rutnät anpassat för pärlplattor.
* **Realtidskontroll:** Justera pärlstorleken med ett enkelt reglage för att passa olika plattor (t.ex. MIDI eller MAXI).
* **Grid-vy:** Ett inbyggt rutnät som hjälper dig att räkna pärlorna rätt.
* **Integritet i fokus:** All bildbehandling sker direkt i din webbläsare via HTML5 Canvas.

## 🛠 Så här kommer du igång (lokalt)
Om du vill utveckla vidare på Peggy lokalt:

   ```bash
   git clone [https://github.com/DITT-ANVÄNDARNAMN/peggy.git](https://github.com/DITT-ANVÄNDARNAMN/peggy.git)
   ```

2. **Gå in i mappen:**
   ```bash

   ```

3. Kör appen:
   Öppna index.html direkt i din webbläsare.
